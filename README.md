# 💰 Django Finance Tracker

Empowering you to take control of your personal finances. This **Django-powered** application helps users track their income and expenses through an intuitive, data-driven dashboard.

## 🚀 Key Features
- **Income/Expense Tracking:** Easily log your transactions.
- **Categorization:** Classify your spending into different sections.
- **Dashboard Overview:** Get a birds-eye view of your latest financial activity.
- **Budgeting:** Control your spending with better visibility.
- **Responsive Dashboard:** Optimized for any screen size.
- **Data Insights:** High performance tracking with refined logic.

## 🛠️ Built With
- **Core Framework:** Python, Django
- **Frontend Assets:** CSS3, JavaScript (Dashboard UI focus)
- **Data Visuals:** Integration of charts and graphs for trends.
- **Database Management:** Supports SQLite, easily portable.

## ⚙️ How to Get Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/yaikobzeray/django-finance-tracker.git
   cd django-finance-tracker
   ```

2. **Virtual Environment Setup:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies:**
   ```bash
   pip install django pillow  # Basic requirements
   ```

4. **Initialize Database:**
   ```bash
   python manage.py migrate
   ```

5. **Fire up the server:**
   ```bash
   python manage.py runserver
   ```

Open `http://127.0.0.1:8000` to start tracking your money!
